## 📊 Phase 2: Power BI Visualization
After analyzing the data with SQL, I built an interactive dashboard in Power BI to make the insights accessible to non-technical users.

### Key Visuals:
* **Suburb Price Map:** A geospatial view of Perth showing that coastal suburbs like Cottesloe command the highest premiums.
* **Average Price Ranking:** A bar chart comparing suburbs to identify market entry points.
* **Bedroom Slicer:** An interactive filter allowing users to see house price trends based on the number of bedrooms.

### Why Power BI?
While SQL is great for deep data mining, Power BI allows for **Data Storytelling**. It turns cold numbers into visual trends that help stakeholders make faster, more informed property decisions.
